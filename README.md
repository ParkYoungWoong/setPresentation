# Make presentations

This was made with ['Reveal.js'](https://github.com/hakimel/reveal.js/).

## CREATE PAGE

```html
<div class="reveal">
  <div class="slides">

    <!-- GENERAL SINGLE PAGE -->
    <section>
    </section>

    <!-- MARKDOWN PAGE -->
    <section data-markdown>
      <textarea data-template>
      </textarea>
    </section>

    <!-- VERTICAL MOVEMENT PAGE -->
    <section>
      <!-- V PAGE 1 -->
      <section></section>
      <!-- V PAGE 2 -->
      <section></section>
      <!-- V PAGE 3 -->
      <section></section>
    </section>

  </div>
</div>
```
