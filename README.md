# Make presentations

This was made with [Reveal.js](https://github.com/hakimel/reveal.js/).

## Start Code

```html
<div class="reveal">
  <div class="slides">

    <section data-markdown>
      <textarea data-template>

      </textarea>
    </section>

  </div>
</div>
```

## How to use

```html
<div class="reveal">
  <div class="slides">


    <!-- GENERAL SINGLE PAGE -->
    <section>
    </section>


    <!-- MARKDOWN PAGE -->
    <section data-markdown>
      <textarea data-template>
        <!-- Enter the Markdown code -->
      </textarea>
    </section>


    <!-- VERTICAL MOVEMENT PAGE -->
    <section>
      <!-- V PAGE 1 -->
      <section>
      </section>
      <!-- V PAGE 2 -->
      <section>
      </section>
    </section>


  </div>
</div>
```
